Brief

We were asked to build a mobile app using React Native and a NodeJS back end. Pledge is presently in beta testing and will be publicly released later this year. Data analysis and storage are critical factors.

Process

We provided architectural plans and development for the mobile app. The app is a React Native implementation targeting Android and iOS. The business critical portions of this app include ACID compliant data storage, data backups, data analysis, and geographic analysis. Additionally, we set up a continuous delivery pipeline for end users and optimized the React Native build process.

Result

Pledge is in beta testing and we look forward to seeing it take flight on mobile devices. We are proud of our work in geographic analysis and continuous delivery, as well as high scale architecture.
